## Bash-Fundamentals
## What is bash?

Bash is a **command line shell** that allows users to interact with the operating system using text commands.
It is used for scripting, automation and managing files

## How is repository helps me learn bash 

This repository helps me learn bash by organizing each concept into its **own script.** 
Each file includes **examples** **explanations** and **troubleshooting notes.** 

## Importance of Documentation

Documentation explains how **code works** so that other people (like me) can understand it.
Without documentation, it can be hard to remember why certain decisions were made or how a script is suppose to run. 
In this repository , i use comments inside each bash file to explain what the code is doing, how it works and what common mistakes to avoid.
This makes each script not just code, but also a learning resource.
Documentation is also useful for debugging. If something breaks, I can read my own explanations and quickly figure out and fix whats wrong instead of guessing. 

## Purpose of Git  

Git is a version conntrol system that tracks changes in code over time. Instead of saving multiple copies of files, Git keeps a history of all changes,
Which makes it easier to manage projects.
In this assignment, Git helps me show my progress ste by step. Each commits represents a small change, such as adding comments or adding a few lines of code.
This makes my learning process visible.
Git also allows me to go back to previous versions if i make a mistake. This is useful when experimenting with new ideas because i can always undo the changes. 

## Is Git Important in the Industry

Git is edtremely important in the tech industry. Almost all software development teams use Git to manage their code and collaborate with others.
It allows multiple developers to work on the same project without overwriting each other's work.
Features like branches and pull requests make teamwork more organized amd efficient.
Even for individual developers, Git is useful beacuse it keeps a full history of work. This makes it easier to track progress, fix bugs and maintain projects over time.
Because of this, learning Git is an important skill for future careers in programming and IT.
