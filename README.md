## Bash-Fundamentals
## What is bash?

Bash is a **command line shell** that allows users to interact with the operating system using text commands.
It is used for scripting, automation and managing files

## How is repository helps me learn bash 

This repository helps me learn bash by organizing each concept into its **own script.** 
Each file includes **examples**, **explanations** and **troubleshooting notes.** 

## Importance of Documentation

Documentation explains how code works so that other people (like me) can understand it.
Without documentation, it can be hard to remember why certain decisions were made or how a script is suppose to run. 
In this repository , i use comments inside each bash file to explain what the code is doing, how it works and what common mistakes to avoid.
This makes each script not just code, but also a learning resource.
Documentation is also useful for debugging. If something breaks, I can read my own explanations and quickly figure out and fix whats wrong instead of guessing.
